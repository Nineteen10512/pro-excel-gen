# Pro Excel Gen v1.3.1

> AI-powered spreadsheet generator — data analysis, charting, Gantt charts, KPI dashboards, and ML integration.

**Pro Excel Gen** 是一款专业级 AI 表格生成技能，支持 Excel/CSV 创建、美化、数据分析、图表生成、甘特图、KPI 仪表盘等全链路表格处理。

---

## ✨ 核心功能 | Core Features

| 功能 | Feature | 说明 |
|------|---------|------|
| 📊 表格生成 | Spreadsheet Generation | 从自然语言描述一键生成结构化 Excel 表格 |
| 🎨 表格美化 | Styling & Formatting | 自动应用专业配色、边框、字体、条件格式 |
| 📈 数据分析 | Data Analysis | pandas 驱动的描述性统计、透视表、相关性分析 |
| 📉 图表生成 | Chart Generation | 柱状图、折线图、饼图、散点图等多种图表 |
| 📅 甘特图 | Gantt Charts | 项目进度甘特图自动生成 |
| 🎯 KPI 仪表盘 | KPI Dashboards | 关键指标仪表盘，一目了然 |
| 🧮 公式支持 | Formula Support | 自动填充 Excel 公式、函数 |
| 🤖 ML 集成 | Machine Learning | pandas/scikit-learn 集成，支持回归、分类建模 |
| 🌐 多格式 | Multi-Format | 支持 xlsx、csv、tsv 等格式读写 |

## 🚀 核心优势 | Key Advantages

- **自然语言即表格**：说人话就能生成专业表格，无需手动操作 Excel
- **分析一站式**：从数据清洗到可视化到建模，全流程在技能内闭环
- **专业颜值**：内置多种配色方案，生成的表格直接可用于汇报
- **项目管理利器**：甘特图 + KPI 仪表盘，项目管理可视化从未如此简单
